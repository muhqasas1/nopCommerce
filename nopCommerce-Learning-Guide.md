## Learning Philosophy

This guide follows the **"Real Application → Concept → Explanation → Code Walkthrough → Business Reasoning"** approach:

1. **Start with Real Code**: We examine actual nopCommerce implementations
2. **Extract the Concept**: Understand the underlying pattern or principle
3. **Deep Dive**: Explain how and why it works
4. **Connect to Business**: Understand the business reasoning behind technical decisions
5. **Practice**: Hands-on exercises to reinforce learning

---

## Prerequisites

- Basic knowledge of C# and .NET
- Understanding of ASP.NET Core fundamentals (controllers, dependency injection, middleware)
- Familiarity with Entity Framework Core
- Basic understanding of design patterns

---

## How to Use This Guide

1. **Read the Code First**: Each chapter starts with actual nopCommerce code
2. **Follow the Path**: The learning path is progressive - each chapter builds on the previous
3. **Do the Exercises**: Complete the practical exercises to reinforce learning
4. **Explore Further**: Use the "Further Reading" sections to dive deeper
5. **Apply Patterns**: Try implementing similar patterns in your own projects

---

## Code Navigation Tips

When exploring nopCommerce code:
- **Libraries**: Core business logic and domain models
- **Nop.Web.Framework**: Web-specific infrastructure and utilities
- **Services**: Business logic implementation
- **Infrastructure**: Cross-cutting concerns and configuration
